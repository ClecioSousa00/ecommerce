## Projeto

Este Projeto foi feito para fins de estudos e por isso ele irá receber funcionalidades, refatoração de código e Testes. Este Ecommerce usa os produtos que foram salvos no supabase, e para fazer as consultas estou usando Prisma e também foi feita a integração com o Stripe para gerecenciar o pagamento dos produtos. Além disso temos a autenticação com o google.
